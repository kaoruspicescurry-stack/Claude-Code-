# Claude Code入門 — コンテンツパイプライン

YouTube動画コンテンツパイプライン — NotebookLM × コメント分析による構成案自動生成

## 概要

3本のClaude Code入門系YouTube動画を分析し、コメントから視聴者インサイトを抽出、動画のトランスクリプトから構成パターンと技法を逆解析。両ブランチの結果を合流させて、最適な動画構成案を一発生成したレポジトリ。

1. [【非エンジニア向け】一番カンタンなClaude Codeの始め方](https://youtu.be/0jKHBND-auw) — AIでサボろうチャンネル
2. [【超入門】初心者でもスグに使えるClaude Codeの基礎と神活用を徹底解説するで！](https://youtu.be/F6GHmnOuQZc) — 【さき】のAIでええやん。
3. [【2026年春最新版】Claude Code入門ガイド](https://youtu.be/-xla8NqooJU) — AI大学

## ファイル構成

| ファイル | 内容 |
|---------|------|
| `Claude_Code入門_構成案.md` | タイトル案・サムネ案・パート構成の完全な構成案 |
| `Claude_Code入門_台本.md` | 構成案を基に作成した動画台本 |

## パイプライン

```
参考動画URL → NotebookLMソース投入 → [ブランチA] コメント抽出→インサイト分析
                                    → [ブランチB] 動画トランスクリプト→逆解析
                                    → [合流] 構成案一発生成
```
